## Project Overview
This is a full-stack routine/timetable generator for Shiv Nadar University (SNU) departments. The frontend provides an interactive UI for inputting courses, faculty, rooms, and constraints; the backend handles API logic and MongoDB persistence; and infrastructure supports local Docker development plus scalable Kubernetes on AWS. ## Technology Stack
- **Frontend**: Next.js 16.2.1, React 18.2.0, TypeScript 5.4.5, Tailwind CSS 3.4.4, TanStack Query, Zustand 4.5.2, jsPDF/html2canvas for exports.
- **Backend**: Node.js/Express 4.21.2, Mongoose 8.20.1 for MongoDB 7.
- **Database**: MongoDB with initialization script for users/roles.
- **DevOps**: Docker Compose (local), Kubernetes (k8s manifests), Terraform (AWS EKS/VPC/ECR), scripts for deploy/backup. ### Step 1: Clone and Navigate
```bash
cd /path/to/snu-routine-generator-02/routineops-platform
```

### Step 2: Configure .env
Create `.env` with:
```
MONGO_ROOT_USER=admin
MONGO_ROOT_PASS=securepass123
JWT_SECRET=your-super-secret-jwt-key-min32chars
NEXT_PUBLIC_API_URL=http://localhost:5000
CORS_ORIGIN=http://localhost:3000
FRONTEND_PORT=3000
NODE_ENV=development
```
The docker-compose.yml uses these vars for service linking. - Frontend depends on backend health, maps port 3000 to host.
